Project Structure of: abhiprojectz/SingularGPT

  • SingularGPT Automated AI toolkit for device control via NLP
    • Prompts
      • prompts.txt Input query, patience for releases
    • README.md Open-source AI, automation, and GPT for device control via NLP.
    • SGPT_Quickstart.txt SGPT: Libraries, repo clone, process 3 init, task_3, multiprocessing, prompt file.
    • component_detection Detects and processes components in images via image processing tools.
      • ip_region.py Detects and preprocesses UI components in an image.
      • lib_ip Image processing tools for component detection and visualization
        • Bbox.py Bounding box image processing class
        • Component.py Component class for shape detection and line identification.
        • file_utils.py Saves detected component coordinates, creates thumbnails.
        • ip_detection.py Python script for detecting and merging image components.
        • ip_draw.py Draws lines and components' boundaries on an image.
        • ip_preprocessing.py Image preprocessing Python script for resizing, grayscaling, filtering, binarization.
    • config Customizable platform and OCR settings, adjustable ZEXUI thresholds and COLOR dictionary.
      • CONFIG.py Platform-specific settings and OCR options
      • CONFIG_ZEXUI.py ZEXUI config adjustable thresholds, COLOR dictionary.
    • example_generated_script.py ZexUI script: writes, waits, clicks, moves, closes.
    • instructions_lib Process and generate commands, ZexUI instructions, OpenAI completions
      • fetch_commands.py Fetch commands from "prompts.txt" in UTF-8 mode, 2 folders up.
      • generate_commands.py Generate OpenAI completions using API.
      • process_instructions.py Generate ZexUI instructions script in parent directory.
    • libs_installation.sh Install libraries, elevated privileges, image processing.
    • main.py Main.py: Imports, generates commands, script generation, and subprocess execution. Unused function call.
    • requirements.txt Python project dependencies: PaddleOCR, OpenCV, Google Cloud Vision API, NumPy, Matplotlib, Xlib, PyVirtualDisplay
    • script.py Scroll, click image, find text, double-left click.
    • text_detection Text detection, OCR integration, noise removal, merged texts, sentence recognition.
      • Text.py Text class: Word width, location detection, image visualization
      • ocr.py OCR script, creates Vision API request for text detection.
      • text_detection.py Text detection with Google or Paddle OCR, noise removal, merged texts, sentence recognition.
    • vision
      • vision_utils.py Text detection and image processing utilities
    • x11_server
      • start.py X11 server: set up, resize, and start display.
    • zexui_lib OpenCV Python library with image, OCR, object detection, error handling.
      • base_functions.py Python image template matching function using numpy and opencv
      • py_gui_commands.py Python GUI commands: mouse actions, screenshots, text input.
      • x11_commands.py Precise mouse, keyboard inputs in Linux with xdotool.
      • zexui.py OpenCV Python library: image, OCR, object detection, error handling.
